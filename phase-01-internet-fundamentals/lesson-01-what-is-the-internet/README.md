# Lesson 1 — What is the Internet?

## Introduction

The Internet is a global network of interconnected networks that allows devices to communicate using standardized protocols, primarily the TCP/IP protocol suite.

## Problem Statement

Computers and networks need a common way to exchange data across different locations, vendors, and infrastructure providers.

## Why This Exists

Direct device-to-device communication does not scale. The Internet solves large-scale communication by connecting networks through routers, ISPs, and shared protocols.

## Architecture View

- User device
- Home/office network
- ISP
- Internet backbone
- Server-side application stack

## Key Concepts

- Network
- Interconnected networks
- Packet switching
- Protocols
- IP addresses

## Real MERN Relevance

Every React fetch request, Node.js API call, webhook, login request, and database interaction depends on Internet communication.

## Takeaways

- The Internet is not the same as the World Wide Web.
- Data travels in packets.
- Standard protocols make global communication possible.
