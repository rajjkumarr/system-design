# Revision — Lesson 1

## Must Remember

- The Internet is a network of networks.
- Data moves in packets.
- TCP/IP is the core protocol suite.
- WWW is a service on top of the Internet.
- Domain names exist because humans remember names better than IP addresses.

## Quick Self-Check

- Can I define the Internet in one sentence?
- Can I explain packet switching in my own words?
- Can I describe a simple MERN request flow?
