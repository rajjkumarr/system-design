# Notes — Lesson 1

## Definition

Internet = global network of interconnected networks using standardized communication protocols.

## Why packet switching matters

- Efficient use of bandwidth
- Retransmit only lost packets
- Better sharing between multiple users
- More resilient than a single large transfer

## Difference between Internet and WWW

- Internet: the infrastructure
- WWW: a service on top of the Internet using HTTP/HTTPS

## MERN request flow

Browser -> DNS -> network -> server -> response -> browser rendering
